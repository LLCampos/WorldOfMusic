About

This app was developed by Luís Campos, Ana Pena and Patrícia Abrantes as a project for the Web Applications course at FCUL. This app uses a RESTful Web Service created by us whose documentation can be read here.

Data Sources

The names of the artists that were used to initially fill our database were obtained using the Lastfm API and the DBPedia SPARQL Endpoint. The information about the artists (with the exception of the Facebook page ID and number of likes and Youtube video ID) is also obtained using the Lastfm API.

The Facebook pages IDs of the artists and number of likes of that pages are obtained using the Facebook Graph API

The Youtube videos IDs are obtained using the Youtube API and the name of the song of the video using, once again, the Lastfm API.

The information about the music of each country was obtained using the MediaWiki API.

This list of countries and all the other information about the countries were obtained using the REST Countries API
